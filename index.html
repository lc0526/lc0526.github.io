使用hexo搭建一个部署在github上的博客

发表于 2015-09-10

一、基础命令：

  • npm install hexo-cli -g // 本地安装hexo 环境
  • hexo init blog // 新建hexo 项目
  • cd blog
  • npm install // 安装依赖包
  • hexo server // 本地运行，端口是4000

  • hero clean
  • hexo generate
  • hexo deploy
